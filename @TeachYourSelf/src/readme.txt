Prog1Tools - beginner's library for Java

              (c) 1999 by Jens Scheffler
****************************************

This archive contains the source-code for the
Prog1Tools library which is used in the book
"Grunkurs Programmieren in Java, Band 1". It
contains classes that are required to work
on the book's exercises. The best-known class
among them is the IOTools-class, which contains
a set of static methods for reading values from
standard input.
